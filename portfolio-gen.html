🔴 특급보안사항
공유 컴퓨터 또는 공용 네트워크에서 본 앱을 사용할 경우, 기입된 AI API Key가 노출되어 보안이 훼손될 수 있습니다. API Key는 브라우저의 localStorage에 저장되며, 같은 브라우저를 사용하는 다른 사용자가 접근할 수 있습니다.

반드시 개인 컴퓨터에서만 사용하세요.
📋 보안정책
1. 본 앱은 모든 데이터를 사용자의 브라우저(localStorage)에만 저장하며, 외부 서버로 전송하지 않습니다.
2. AI 기능 사용 시, 입력된 텍스트가 선택한 AI 서비스(OpenAI, Google, Anthropic)의 API로 전송됩니다.
3. AI API Key는 암호화되지 않은 상태로 localStorage에 저장됩니다.
4. 브라우저 데이터 삭제 시 모든 설정 및 API Key가 함께 삭제됩니다.
5. 본 앱의 소스 코드는 사용자의 로컬 환경에서만 실행됩니다.
📋 면책조항
1. 본 앱 사용으로 인해 발생하는 데이터 유출, API Key 도용 등의 피해에 대해 개발자는 책임을 지지 않습니다.
2. AI가 생성한 텍스트의 정확성, 적절성에 대한 책임은 사용자에게 있습니다.
3. 본 앱은 교육 및 개인 포트폴리오 제작 목적으로 제공되며, 상업적 용도의 사용은 권장하지 않습니다.
4. 제출물에 대한 학술적 책임은 전적으로 사용자에게 있습니다.
